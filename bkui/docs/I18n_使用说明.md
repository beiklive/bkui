# I18n 国际化使用说明

## 概述

BeikUI 提供基于 JSON 键值对的国际化（i18n）支持，通过 C++ UDL（User-Defined Literal）实现简洁的翻译 API。

核心组件：
- `bk::I18n` — 翻译管理器（单例），负责加载翻译 JSON、探测系统语言
- `""_i18n` — UDL 字面量操作符，将字符串键转换为可翻译的代理对象
- `resources/i18n/*.json` — 翻译资源文件

## 快速开始

### 1. 引入头文件

```cpp
#include <bkui/bkui.hpp>  // 已包含 <bkui/core/I18n.hpp>
```

### 2. 初始化

```cpp
bk::FileSystem::Mount("resources");
bk::I18n::Instance().Init("i18n");  // 加载 resources/i18n/ 下的翻译文件
```

Init 会自动通过 SDL3 的 `SDL_GetPreferredLocales()` 探测系统语言，并依次回退匹配：
- 地区匹配（如 `zh-Hans-CN`）→ 语言匹配（如 `zh`）→ 特殊回退（`zh` 尝试 `zh-CN`）→ 默认 `en`

### 3. 使用翻译

在所有需要翻译的字符串处使用 `""_i18n` 字面量：

```cpp
using bk::operator""_i18n;

// 无参翻译：返回代理对象，隐式转换为 std::string (UTF-8)
label->SetText("settings/title"_i18n);

// 带参数翻译：{} 占位符按出现顺序填入
label->SetText("settings/language"_i18n("English"));
label->SetText("svg/rotation"_i18n(45.0f, 1.05f));  // → "旋转: 45°  缩放: 1.05"

// 也可显式调用 .str() 或 .u32()
std::string s = "key"_i18n;              // 隐式转换
std::string s = "key"_i18n.str();        // 显式转换
std::u32string u = "key"_i18n.u32();     // 转为 u32string
```

## 翻译文件格式

翻译文件位于 `resources/i18n/` 目录，格式为 JSON：

**资源文件结构：**
```
resources/i18n/
  en.json         # 英文
  zh-CN.json      # 简体中文
  ja.json         # 日文
  ko.json         # 韩文
```

**JSON 格式：**
```json
{
    "settings/title": "Settings",
    "settings/language": "Language: {}",
    "svg/rotation": "Rotation: {}°  Scale: {}x"
}
```

- **Key**: 与代码中的 `""_i18n` 字面量一致
- **Value**: 对应语言的翻译文本
- **占位符**: `{}` — 匿名占位符，按调用时参数顺序填入

### Key 命名建议

使用 `/` 分隔的层级命名，如：
- `modal/title` — 模态弹窗标题
- `nav/home` — 导航按钮
- `app/title` — 应用标题
- `status_panel/tip` — 状态面板提示

## API 参考

### `""_i18n` UDL

```cpp
// 创建翻译代理对象
"key"_i18n                    // → I18nString
"key"_i18n(arg1, arg2, ...)   // → I18nString (带参数)
```

返回的 `I18nString` 对象提供两种输出方法：

| 方法 | 返回类型 | 适用场景 |
|------|---------|---------|
| 隐式转换 | `std::string` (UTF-8) | `Label::SetText("key"_i18n)` |
| `.str()` | `std::string` (UTF-8) | 显式转换，同隐式 |
| `.u32()` | `std::u32string` | `DrawText(..., "key"_i18n.u32(), ...)` |

### `bk::I18n` 类

```cpp
class I18n
{
public:
    static I18n& Instance();

    void Init(std::string_view resourceDir = "i18n");
    void SetLanguage(std::string_view langCode);

    std::string_view CurrentLanguage() const;
};
```

| 方法 | 说明 |
|------|------|
| `Instance()` | 获取单例 |
| `Init(resourceDir)` | 加载翻译文件，自动探测系统语言 |
| `SetLanguage(lang)` | 运行时切换语言（不会自动刷新 UI） |
| `CurrentLanguage()` | 获取当前语言代码 |

### 参数类型支持

`""_i18n(arg)` 支持的参数类型：

| 类型 | 格式化方式 |
|------|-----------|
| `int` | `std::to_string()` |
| `float` | `std::format("{:.2f}")` (保留2位小数) |
| `double` | `std::format("{:.2f}")` |
| `const char*` | 直接拼接 |
| `std::string` | 直接拼接 |
| `std::string_view` | 直接拼接 |

### Fallback 行为

- Key 在翻译文件中不存在 → 直接返回 key 原文
- 翻译文件不存在 → 返回 key 原文
- `Init()` 未调用 → 所有 `_i18n` 返回 key 原文

## 完整示例

```cpp
#include <bkui/bkui.hpp>

using bk::operator""_i18n;

void BuildUI()
{
    auto& i18n = bk::I18n::Instance();

    // 初始化
    bk::FileSystem::Mount("resources");
    i18n.Init("i18n");

    // 使用翻译
    auto titleLabel = std::make_shared<bk::Label>("app/title"_i18n);
    auto langLabel = std::make_shared<bk::Label>(
        "settings/language"_i18n(i18n.CurrentLanguage()));

    // 运行时切换语言
    i18n.SetLanguage("zh-CN");
    // 切换后需要重新设置 UI 文本
    titleLabel->SetText("app/title"_i18n);
}
```

## 添加新语言

1. 在 `resources/i18n/` 目录创建新 JSON 文件（如 `ja.json`）
2. 复制 `en.json` 的 key 结构
3. 翻译所有 value 为对应语言
4. 无需重新编译，启动时自动加载

> 注意：`resources/i18n/` 目录由 CMake 自动复制到可执行文件旁
