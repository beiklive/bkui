# BeikUI 项目搭建实施文档（Codex执行最终版 v3）

## 1. 项目目标

BeikUI 是一个跨平台 C++20 UI 引擎。

目标：

* Windows
* Linux
* Android
* Nintendo Switch
* OpenGL优先开发
* 后续支持Deko3D
* 后续支持Vulkan
* 支持模拟器前端
* 支持游戏手柄导航
* 支持JSON描述式UI
* 支持Web编辑器生成UI
* 支持热更新
* 支持统一资源系统
* 支持资源包
* 可编译为静态库和动态库
* UI层禁止调用图形API

命名：

项目：

```text
BeikUI
```

库：

```text
bkui
```

命名空间：

```cpp

{
}
```

工具：

```text

```

资源包：

```text
.bkres
```

UI编译资源：

```text
.bkui
```

架构：

```text

↓

↓

↓

↓

↓

System
```

```cmake
    CMAKE_CXX_STANDARD
```

## 4. 库构建

bkui/CMakeLists：

```cmake
option(
    BKUI_SHARED
    "Build shared library"
    OFF
)

file(
    GLOB_RECURSE
    BKUI_SRC
    src/*.cpp
)

if(BKUI_SHARED)

add_library(
    bkui
    SHARED
    ${BKUI_SRC}
)

else()

add_library(
    bkui
    STATIC
    ${BKUI_SRC}
)

endif()

target_include_directories(
    bkui
    PUBLIC
    include
)

target_compile_features(
    bkui
    PUBLIC
    cxx_std_20
)
```

## 5. Platform层

统一平台接口：

```cpp
namespace bk
{

class Platform
{
public:

    virtual bool Init()=0;

    virtual void PollEvents()=0;

    virtual void Shutdown()=0;

    virtual WindowHandle GetWindow()=0;

    virtual InputState GetInput()=0;

    virtual Vector2 GetWindowSize()=0;

};

}
```

实现：

```text
platform/

├── Platform.hpp
├── SDLPlatform.cpp
├── SwitchPlatform.cpp
└── AndroidPlatform.cpp
```

平台实现：

```text
Windows
Linux
Android

↓

SDL3


Nintendo Switch

↓

libnx
```

Switch禁止：

```cpp
SDL_Init()
```

Switch使用：

```cpp
appletMainLoop();

hidScanInput();

```

```text

↓

↓

↓
```

```cpp

{

class FileSystem
{
public:

    static void Mount(
        const std::string&
    );

    static Buffer Read(
        const std::string&
    );

};

}
```

```cpp
PHYSFS_openRead()
```
