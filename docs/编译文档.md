1. 编译需要确保windows平台和switch平台都能编译通过，switch平台需要编译出nro文件

2. switch平台编译使用 "E:\bin\msys64\msys2.exe" 进行编译
如果缺少依赖库，在安装前需要配置代理：

```bash
# 代理设置
export http_proxy="http://127.0.0.1:7892"
export https_proxy="http://127.0.0.1:7892"
```

3. 编译需要制作成编译脚本，直接放在项目根目录，分为windows平台和switch平台两个脚本，分别为build_windows.bat和build_switch.sh
